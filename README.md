## About

IQVerse is a collection of modern, browser-based web apps and developer tools. Every experience is:

- **100% Client-Side** – Your data never leaves your browser
- **Open Source** – MIT licensed, source code on GitHub
- **Privacy-First** – No tracking, no telemetry, no accounts needed
- **Developer-Friendly** – Built by developers, for developers

## 🚀 Getting Started

All features run entirely in your browser. No installation required.

1. **Open the app** – Visit the homepage to get started
2. **Use it immediately** – No login, no setup
3. **Your data stays private** – Everything is processed locally

### Local development

If you want to run the site locally:

```bash
yarn dev
```

To create a production build:

```bash
yarn build
```

The project is built as a static Next.js site and is ready for Cloudflare Pages deployment.

### Project structure

- `app/` contains the App Router pages and tool routes
- `components/` holds reusable UI and tool components
- `styles/` contains CSS modules for the interface
- `public/` stores static assets such as icons and manifests
